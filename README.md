# MacOS-Tools

A collection of macOS-focused utilities.

## Included Projects

### netmon

`netmon` is a high-performance CLI network monitor for macOS, inspired by heavy daily use of `bmon`, but built with additional features and a faster refresh cadence.

- Path: [`netmon/`](netmon)
- Details: [`netmon/README.md`](netmon/README.md)
